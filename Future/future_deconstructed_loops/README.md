# Future Non-Static Deconstructed Loops

Built as an evolving futuristic deconstructed loop pack.

## Included
- `full_mix/` full reference mix
- `stems/` six separated evolving stems
- `4_bar_variants/` shorter playable extractions
- `generation_notes.py`

## Core rule
These are **not** static loops.
The internal structure changes across the timeline instead of simply repeating the same bar.
