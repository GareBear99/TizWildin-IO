# TizWildin Wavetable Synth Loops Pack 01

A custom pack of seamless wavetable-synth loops rendered at **48000 Hz / 16-bit WAV**.

## Included
- **TW_WavetableLoop_01_CelestialMotion_124BPM.wav** — lush major-7 motion bed with evolving vocal-saw morph
- **TW_WavetableLoop_02_NeonDrift_124BPM.wav** — tighter rhythmic synth loop with glassy top end
- **TW_WavetableLoop_03_ObsidianPulse_124BPM.wav** — dark pulsing arpeggiated wavetable synth loop
- **TW_WavetableLoop_04_AuroraBloom_124BPM.wav** — wide airy slow-bloom wavetable progression

## Specs
- Tempo: **124 BPM**
- Length: **8 bars each**
- Designed to loop cleanly
- No drums included
- Wavetable-based synthesis with light stereo motion and circular delay

## Notes
These are production-oriented synth loops meant to be dropped directly into a DAW or sampler.
A generation script is included so the pack can be regenerated and extended.
